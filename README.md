[ INSTALLING NODEJS ]

```bash
ubuntu
curl -fsSL https://deb.nodesource.com/setup_current.x | sudo -E bash -
apt install -y nodejs gcc g++ make
git clone https://github.com/ikyalwaysgood/KinaBot-MD
cd KinaBot-MD
npm install
```
